## 🎯 Stage 1: The Core Idea

### What Happens When an Electrolyte Dissolves?

When ionic compounds dissolve in water, they **dissociate** into their constituent ions. The concentration of each ion depends on the formula of the salt.

> **Analogy:** Imagine a bag of marbles — each bag contains 1 red and 2 blue marbles (like CaCl₂ → Ca²⁺ + 2Cl⁻). If you open 10 bags, you get 10 red and 20 blue marbles. The formula tells you the "marble ratio."

### The Rule

```
If a salt A_x B_y dissolves to give x ions of A and y ions of B:

    A_x B_y → x A^(y+) + y B^(x-)

Then:
    [A] = x × [salt]
    [B] = y × [salt]
```

### Strong vs. Weak Electrolytes

| Type | Dissociation | Example |
|------|-------------|---------|
| **Strong electrolyte** | 100% (complete) | NaCl, HCl, NaOH, KNO₃ |
| **Weak electrolyte** | Partial (use α) | CH₃COOH, NH₃, H₂CO₃ |

> For this chapter, we focus primarily on **strong electrolytes** (complete dissociation). Weak electrolytes are covered in Type 9.

## 🔬 Stage 2: The Formula Lab

### The Formula

```
For salt A_x B_y of concentration C (mol/L):

    A_x B_y → x A^n+ + y B^m-

    [A^n+] = x × C
    [B^m-] = y × C
    Total ion concentration = (x + y) × C
```

### Important Dissociation Examples

| Salt | Dissociation | [Cation] | [Anion] | Total ions |
|------|-------------|----------|---------|------------|
| NaCl | Na⁺ + Cl⁻ | C | C | 2C |
| CaCl₂ | Ca²⁺ + 2Cl⁻ | C | 2C | 3C |
| Al₂(SO₄)₃ | 2Al³⁺ + 3SO₄²⁻ | 2C | 3C | 5C |
| Na₃PO₄ | 3Na⁺ + PO₄³⁻ | 3C | C | 4C |
| K₂SO₄ | 2K⁺ + SO₄²⁻ | 2C | C | 3C |
| Fe₂(SO₄)₃ | 2Fe³⁺ + 3SO₄²⁻ | 2C | 3C | 5C |

### Charge Balance Principle

```
In any solution:
    Total positive charge = Total negative charge

    Σ (charge × concentration) of cations = Σ (charge × concentration) of anions

This ALWAYS holds. It's not a formula — it's a law of nature (electroneutrality).
```

### Type 2: Polyatomic Salts (1:2 or 2:1 ratio) ⭐

**The Pattern:** Salts like CaCl₂, Na₂SO₄ → unequal number of cations and anions.

#### Solved Example 10.2
**Q:** Find the concentration of each ion in 0.1 M CaCl₂. 🟢 ⭐

**Solution:**
```
    CaCl₂ → Ca²⁺ + 2Cl⁻

    [Ca²⁺] = 1 × 0.1 = 0.1 M
    [Cl⁻]  = 2 × 0.1 = 0.2 M

    Total ion concentration = 0.1 + 0.2 = 0.3 M

Charge check:
    Positive: 0.1 × 2 = 0.2
    Negative: 0.2 × 1 = 0.2  ✅ (balanced!)

Answer: [Ca²⁺] = 0.1 M, [Cl⁻] = 0.2 M
```

#### Solved Example 10.2B
**Q:** Find all ion concentrations in 0.2 M Na₂SO₄. 🟢

```
    Na₂SO₄ → 2Na⁺ + SO₄²⁻

    [Na⁺]   = 2 × 0.2 = 0.4 M
    [SO₄²⁻] = 1 × 0.2 = 0.2 M

Answer: [Na⁺] = 0.4 M, [SO₄²⁻] = 0.2 M
```

### Type 4: Total Ion Concentration

**The Pattern:** Sum of ALL ion concentrations.

#### Solved Example 10.4
**Q:** Which 0.1 M solution has the highest total ion concentration? (a) NaCl (b) CaCl₂ (c) Al₂(SO₄)₃ 🟡

**Solution:**
```
(a) NaCl → Na⁺ + Cl⁻             → 2 ions per formula unit
    Total = 2 × 0.1 = 0.2 M

(b) CaCl₂ → Ca²⁺ + 2Cl⁻          → 3 ions per formula unit
    Total = 3 × 0.1 = 0.3 M

(c) Al₂(SO₄)₃ → 2Al³⁺ + 3SO₄²⁻  → 5 ions per formula unit
    Total = 5 × 0.1 = 0.5 M

Answer: Al₂(SO₄)₃ has the highest total ion concentration.
```

> **Connection to colligative properties:** Colligative properties depend on total number of particles. So 0.1 M Al₂(SO₄)₃ would have a greater boiling point elevation than 0.1 M NaCl (van't Hoff factor i = 5 vs i = 2).

### Type 5: Mixed Electrolytes — Common Ion ⭐⭐

**The Pattern:** Two different salts contribute the same ion → concentrations add up.

#### Solved Example 10.5
**Q:** 0.1 M NaCl and 0.2 M KCl are mixed in equal volumes. Find the concentration of each ion. 🟡 ⭐⭐

**Solution:**
```
Step 1: After mixing equal volumes, each concentration halves
    [NaCl]_eff = 0.1/2 = 0.05 M
    [KCl]_eff = 0.2/2 = 0.1 M

Step 2: Dissociation
    NaCl → Na⁺ + Cl⁻     → [Na⁺] = 0.05 M, [Cl⁻] from NaCl = 0.05 M
    KCl  → K⁺ + Cl⁻      → [K⁺]  = 0.1 M,  [Cl⁻] from KCl = 0.1 M

Step 3: Total Cl⁻ (common ion)
    [Cl⁻] = 0.05 + 0.1 = 0.15 M

Charge check:
    Positive: 0.05(1) + 0.1(1) = 0.15
    Negative: 0.15(1) = 0.15  ✅

Answer: [Na⁺] = 0.05 M, [K⁺] = 0.1 M, [Cl⁻] = 0.15 M
```

> **Key insight:** Cl⁻ comes from BOTH salts. You must add its contributions from each source.

### Type 6: Dilution of Ionic Solution

**The Pattern:** Diluting an ionic solution → all ion concentrations decrease proportionally.

#### Solved Example 10.6
**Q:** 100 mL of 0.5 M CaCl₂ is diluted to 500 mL. Find the new [Ca²⁺] and [Cl⁻]. 🟡

**Solution:**
```
Step 1: Find new [CaCl₂] after dilution
    M₁V₁ = M₂V₂
    0.5 × 100 = M₂ × 500
    M₂ = 0.1 M

Step 2: Ion concentrations
    CaCl₂ → Ca²⁺ + 2Cl⁻
    [Ca²⁺] = 0.1 M
    [Cl⁻] = 0.2 M

Answer: [Ca²⁺] = 0.1 M, [Cl⁻] = 0.2 M
```

> **Shortcut:** Each ion's concentration gets diluted by the same factor. Dilution factor = 500/100 = 5×. Original [Ca²⁺] = 0.5 M → new = 0.5/5 = 0.1 M. Original [Cl⁻] = 1.0 M → new = 1.0/5 = 0.2 M.

### Type 7: Ion Concentration After Mixing Two Electrolyte Solutions ⭐⭐

**The Pattern:** Combine mixing (Ch 9) with dissociation. Two different salts, possibly reacting.

#### Solved Example 10.7
**Q:** 100 mL of 0.2 M NaCl is mixed with 200 mL of 0.1 M Na₂SO₄. Find the concentration of Na⁺, Cl⁻, and SO₄²⁻. 🟡 ⭐

**Solution:**
```
Step 1: Find effective concentrations after mixing
    V_total = 100 + 200 = 300 mL

    [NaCl] = 0.2 × 100/300 = 0.0667 M
    [Na₂SO₄] = 0.1 × 200/300 = 0.0667 M

Step 2: Dissociation
    NaCl → Na⁺ + Cl⁻
        [Na⁺] from NaCl = 0.0667 M
        [Cl⁻] = 0.0667 M

    Na₂SO₄ → 2Na⁺ + SO₄²⁻
        [Na⁺] from Na₂SO₄ = 2 × 0.0667 = 0.1334 M
        [SO₄²⁻] = 0.0667 M

Step 3: Total [Na⁺] (common ion!)
    [Na⁺] = 0.0667 + 0.1334 = 0.2 M

Charge check:
    Positive: 0.2 × 1 = 0.2
    Negative: 0.0667 × 1 + 0.0667 × 2 = 0.0667 + 0.1334 = 0.2  ✅

Answer: [Na⁺] = 0.2 M, [Cl⁻] = 0.067 M, [SO₄²⁻] = 0.067 M
```

### Type 8: Precipitation — Will a Precipitate Form? (Ksp Preview) ⭐

**The Pattern:** When two electrolytes are mixed, an insoluble salt may form. Compare ionic product (Q) with Ksp.

#### Key Concept
```
Ionic Product (Q) = [cation]^a × [anion]^b  (after mixing, before precipitation)

If Q > Ksp → Precipitate forms ✅
If Q < Ksp → No precipitate ❌
If Q = Ksp → Saturated, at equilibrium
```

#### Solved Example 10.8
**Q:** 100 mL of 0.01 M BaCl₂ is mixed with 100 mL of 0.01 M Na₂SO₄. Will BaSO₄ precipitate? (Ksp of BaSO₄ = 1.1 × 10⁻¹⁰) 🟡 ⭐

**Solution:**
```
Step 1: Find ion concentrations after mixing
    V_total = 200 mL (each concentration halves)
    
    [Ba²⁺] = 0.01/2 = 0.005 M
    [SO₄²⁻] = 0.01/2 = 0.005 M

Step 2: Calculate ionic product (Q)
    Q = [Ba²⁺][SO₄²⁻] = 0.005 × 0.005 = 2.5 × 10⁻⁵

Step 3: Compare with Ksp
    Q = 2.5 × 10⁻⁵ >> Ksp = 1.1 × 10⁻¹⁰

    Q >> Ksp → BaSO₄ WILL precipitate! ✅

Answer: Yes, BaSO₄ precipitates (Q is about 10⁵ times larger than Ksp).
```

### Type 9: Weak Electrolytes — Partial Dissociation (α)

**The Pattern:** Weak electrolytes don't fully dissociate. Use degree of dissociation (α).

#### Key Formula
```
For HA ⇌ H⁺ + A⁻  (degree of dissociation = α)

    [H⁺] = C × α
    [A⁻] = C × α
    [HA] = C × (1 - α)

where C = initial concentration, α = fraction dissociated (0 < α < 1)
```

#### Solved Example 10.9
**Q:** 0.1 M acetic acid (CH₃COOH) has α = 0.013 (1.3% dissociation). Find [H⁺], [CH₃COO⁻], and [CH₃COOH]. 🟡

**Solution:**
```
    CH₃COOH ⇌ H⁺ + CH₃COO⁻

    [H⁺] = C × α = 0.1 × 0.013 = 1.3 × 10⁻³ M
    [CH₃COO⁻] = C × α = 1.3 × 10⁻³ M
    [CH₃COOH] = C × (1 - α) = 0.1 × 0.987 = 0.0987 M ≈ 0.1 M

    (Only 1.3% of the acid has dissociated — most remains intact.)

Answer: [H⁺] = [CH₃COO⁻] = 1.3 × 10⁻³ M, [CH₃COOH] ≈ 0.099 M
```

> **Key difference from Types 1-8:** Strong electrolytes: α = 1 (100% dissociation, use stoichiometry directly). Weak electrolytes: α < 1 (need to know α to find ion concentrations).

### Type 10: Charge Balance — Electroneutrality

**The Pattern:** The total positive charge in solution must always equal the total negative charge.

#### Solved Example 10.10
**Q:** A solution contains 0.1 M NaCl and 0.2 M CaCl₂. Verify the charge balance. 🟡

**Solution:**
```
Dissociation:
    NaCl → Na⁺ + Cl⁻          [Na⁺] = 0.1, [Cl⁻] = 0.1
    CaCl₂ → Ca²⁺ + 2Cl⁻      [Ca²⁺] = 0.2, [Cl⁻] = 0.4

Total ions:
    [Na⁺] = 0.1 M
    [Ca²⁺] = 0.2 M
    [Cl⁻] = 0.1 + 0.4 = 0.5 M

Charge balance:
    Total positive charge = [Na⁺](+1) + [Ca²⁺](+2)
                         = 0.1 × 1 + 0.2 × 2 = 0.1 + 0.4 = 0.5

    Total negative charge = [Cl⁻](−1)
                         = 0.5 × 1 = 0.5

    Positive = Negative ✅ (electroneutrality maintained)
```

> **Why charge balance matters:** It's a powerful error-checking tool. If your calculated ion concentrations don't satisfy charge balance, you've made a mistake somewhere.

## Key Takeaways from Chapter 10

```
┌──────────────────────────────────────────────────────────────┐
│              IONIC CONCENTRATION SUMMARY                      │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  Rule: A_xB_y → x A^n+ + y B^m-                             │
│        [A] = x × C,  [B] = y × C                            │
│                                                               │
│  Strong electrolytes: 100% dissociation (α = 1)             │
│  Weak electrolytes: partial (use α)                          │
│                                                               │
│  Common ions: ADD contributions from all sources             │
│  e.g., NaCl + KCl → [Cl⁻] = [NaCl] + [KCl]                │
│                                                               │
│  Charge balance (ALWAYS holds):                               │
│    Σ(cation charge × conc) = Σ(anion charge × conc)         │
│                                                               │
│  Precipitation: Q vs Ksp                                      │
│    Q > Ksp → precipitate forms                               │
│    Q < Ksp → no precipitate                                  │
│                                                               │
│  Total ions per formula unit:                                 │
│    NaCl: 2 | CaCl₂: 3 | Al₂(SO₄)₃: 5                      │
│    → affects colligative properties (van't Hoff factor)      │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```
