## 🎯 Stage 1: The Core Idea

### What Happens When an Electrolyte Dissolves?

When ionic compounds dissolve in water, they **dissociate** into their constituent ions. The concentration of each ion depends on the formula of the salt.

> **Analogy:** Imagine a bag of marbles — each bag contains 1 red and 2 blue marbles (like CaCl₂ → Ca²⁺ + 2Cl⁻). If you open 10 bags, you get 10 red and 20 blue marbles. The formula tells you the "marble ratio."

### The Rule

```
If a salt A_x B_y dissolves to give x ions of A and y ions of B:

    A_x B_y → x A^(y+) + y B^(x-)

Then:
    [A] = x × [salt]
    [B] = y × [salt]
```

### Strong vs. Weak Electrolytes

| Type | Dissociation | Example |
|------|-------------|---------|
| **Strong electrolyte** | 100% (complete) | NaCl, HCl, NaOH, KNO₃ |
| **Weak electrolyte** | Partial (use α) | CH₃COOH, NH₃, H₂CO₃ |

> For this chapter, we focus primarily on **strong electrolytes** (complete dissociation). Weak electrolytes are covered in Type 9.

## 🔬 Stage 2: The Formula Lab

### The Formula

```
For salt A_x B_y of concentration C (mol/L):

    A_x B_y → x A^n+ + y B^m-

    [A^n+] = x × C
    [B^m-] = y × C
    Total ion concentration = (x + y) × C
```

### Important Dissociation Examples

| Salt | Dissociation | [Cation] | [Anion] | Total ions |
|------|-------------|----------|---------|------------|
| NaCl | Na⁺ + Cl⁻ | C | C | 2C |
| CaCl₂ | Ca²⁺ + 2Cl⁻ | C | 2C | 3C |
| Al₂(SO₄)₃ | 2Al³⁺ + 3SO₄²⁻ | 2C | 3C | 5C |
| Na₃PO₄ | 3Na⁺ + PO₄³⁻ | 3C | C | 4C |
| K₂SO₄ | 2K⁺ + SO₄²⁻ | 2C | C | 3C |
| Fe₂(SO₄)₃ | 2Fe³⁺ + 3SO₄²⁻ | 2C | 3C | 5C |

### Charge Balance Principle

```
In any solution:
    Total positive charge = Total negative charge

    Σ (charge × concentration) of cations = Σ (charge × concentration) of anions

This ALWAYS holds. It's not a formula — it's a law of nature (electroneutrality).
```

### Type 1: Simple Salt (1:1 ratio)

**The Pattern:** Salts like NaCl, KBr, NaNO₃ → produce 1 cation and 1 anion.

#### Solved Example 10.1
**Q:** Find the concentration of all ions in 0.1 M NaCl. 🟢

**Solution:**
```
    NaCl → Na⁺ + Cl⁻       (100% dissociation)

    [Na⁺] = 0.1 M
    [Cl⁻] = 0.1 M
    Total ion concentration = 0.1 + 0.1 = 0.2 M

Answer: [Na⁺] = 0.1 M, [Cl⁻] = 0.1 M
```

### Type 2: Polyatomic Salts (1:2 or 2:1 ratio) ⭐

**The Pattern:** Salts like CaCl₂, Na₂SO₄ → unequal number of cations and anions.

#### Solved Example 10.2
**Q:** Find the concentration of each ion in 0.1 M CaCl₂. 🟢 ⭐

**Solution:**
```
    CaCl₂ → Ca²⁺ + 2Cl⁻

    [Ca²⁺] = 1 × 0.1 = 0.1 M
    [Cl⁻]  = 2 × 0.1 = 0.2 M

    Total ion concentration = 0.1 + 0.2 = 0.3 M

Charge check:
    Positive: 0.1 × 2 = 0.2
    Negative: 0.2 × 1 = 0.2  ✅ (balanced!)

Answer: [Ca²⁺] = 0.1 M, [Cl⁻] = 0.2 M
```

#### Solved Example 10.2B
**Q:** Find all ion concentrations in 0.2 M Na₂SO₄. 🟢

**Solution:**
```
    Na₂SO₄ → 2Na⁺ + SO₄²⁻

    [Na⁺]   = 2 × 0.2 = 0.4 M
    [SO₄²⁻] = 1 × 0.2 = 0.2 M

Answer: [Na⁺] = 0.4 M, [SO₄²⁻] = 0.2 M
```

### Type 4: Total Ion Concentration

**The Pattern:** Sum of ALL ion concentrations.

#### Solved Example 10.4
**Q:** Which 0.1 M solution has the highest total ion concentration? (a) NaCl (b) CaCl₂ (c) Al₂(SO₄)₃ 🟡

**Solution:**
```
(a) NaCl → Na⁺ + Cl⁻             → 2 ions per formula unit
    Total = 2 × 0.1 = 0.2 M

(b) CaCl₂ → Ca²⁺ + 2Cl⁻          → 3 ions per formula unit
    Total = 3 × 0.1 = 0.3 M

(c) Al₂(SO₄)₃ → 2Al³⁺ + 3SO₄²⁻  → 5 ions per formula unit
    Total = 5 × 0.1 = 0.5 M

Answer: Al₂(SO₄)₃ has the highest total ion concentration.
```

> **Connection to colligative properties:** Colligative properties depend on total number of particles. So 0.1 M Al₂(SO₄)₃ would have a greater boiling point elevation than 0.1 M NaCl (van't Hoff factor i = 5 vs i = 2).

**10.4a:**
*   **Calculation:** Total ions = $M \times i$ (van't Hoff factor).
    Glucose ($i=1$): $0.1 \times 1 = 0.1\text{ M}$.

    MgSO₄ ($i=2$): $0.1 \times 2 = 0.2\text{ M}$.

*   **Answer:** Glucose ($0.1\text{ M}$) $<$ NaCl ($0.2\text{ M}$) $=$ MgSO₄ ($0.2\text{ M}$) $<$ AlCl₃ ($0.4\text{ M}$)

**10.4b:**
*   **Calculation:** K₂SO₄ yields 3 ions (2K⁺ + SO₄²⁻).
    Total ion concentration = $0.2 \times 3 = 0.6\text{ M}$.
*   **Answer:** $0.6\text{ M}$

**10.4c:**
*   **Calculation:** FeCl₃ yields 4 ions (Fe³⁺ + 3Cl⁻). Assuming it is a strong electrolyte, it dissociates completely (no undissociated FeCl₃).
    Total particle concentration = $0.05 \times 4 = 0.2\text{ M}$.
*   **Answer:** $0.2\text{ M}$

</details>

### Type 5: Mixed Electrolytes — Common Ion ⭐⭐

**The Pattern:** Two different salts contribute the same ion → concentrations add up.

#### Solved Example 10.5
**Q:** 0.1 M NaCl and 0.2 M KCl are mixed in equal volumes. Find the concentration of each ion. 🟡 ⭐⭐

**Solution:**
```
Step 1: After mixing equal volumes, each concentration halves
    [NaCl]_eff = 0.1/2 = 0.05 M
    [KCl]_eff = 0.2/2 = 0.1 M

Step 2: Dissociation
    NaCl → Na⁺ + Cl⁻     → [Na⁺] = 0.05 M, [Cl⁻] from NaCl = 0.05 M
    KCl  → K⁺ + Cl⁻      → [K⁺]  = 0.1 M,  [Cl⁻] from KCl = 0.1 M

Step 3: Total Cl⁻ (common ion)
    [Cl⁻] = 0.05 + 0.1 = 0.15 M

Charge check:
    Positive: 0.05(1) + 0.1(1) = 0.15
    Negative: 0.15(1) = 0.15  ✅

Answer: [Na⁺] = 0.05 M, [K⁺] = 0.1 M, [Cl⁻] = 0.15 M
```

> **Key insight:** Cl⁻ comes from BOTH salts. You must add its contributions from each source.

<summary>💡 Detailed Solutions for Type 5</summary>

**10.5a:**
*   **Calculation:** After mixing giving $V_{\text{tot}} = 200\text{ mL}$:
    $[\text{NaCl}] = \frac{0.2 \times 100}{200} = 0.1\text{ M}$. $[\text{CaCl}_2] = \frac{0.1 \times 100}{200} = 0.05\text{ M}$.
    $[\text{Na}^+] = 0.1\text{ M}$. $[\text{Ca}^{2+}] = 0.05\text{ M}$.
    $[\text{Cl}^-] = [\text{NaCl}] + 2 \times [\text{CaCl}_2] = 0.1 + 2(0.05) = 0.2\text{ M}$.
*   **Answer:** $[\text{Na}^+] = 0.1\text{ M}, [\text{Ca}^{2+}] = 0.05\text{ M}, [\text{Cl}^-] = 0.2\text{ M}$

**10.5b:**
*   **Calculation:** $V_{\text{tot}} = 200 + 300 = 500\text{ mL}$.
    $[\text{Na}_2\text{SO}_4] = \frac{0.3 \times 200}{500} = 0.12\text{ M}$. $[\text{NaCl}] = \frac{0.1 \times 300}{500} = 0.06\text{ M}$.
    $[\text{SO}_4^{2-}] = 0.12\text{ M}$. $[\text{Cl}^-] = 0.06\text{ M}$.
    $[\text{Na}^+] = 2 \times [\text{Na}_2\text{SO}_4] + [\text{NaCl}] = 2(0.12) + 0.06 = 0.30\text{ M}$.
*   **Answer:** $[\text{Na}^+] = 0.3\text{ M}, [\text{SO}_4^{2-}] = 0.12\text{ M}, [\text{Cl}^-] = 0.06\text{ M}$

**10.5c:**
*   **Calculation:** Equal volumes, so concentrations halve:
    $[\text{KNO}_3] = 0.2\text{ M}$. $[\text{Ca(NO}_3)_2] = 0.1\text{ M}$.
    $[\text{K}^+] = 0.2\text{ M}$. $[\text{Ca}^{2+}] = 0.1\text{ M}$.
    $[\text{NO}_3^-] = [\text{KNO}_3] + 2 \times [\text{Ca(NO}_3)_2] = 0.2 + 2(0.1) = 0.4\text{ M}$.
*   **Answer:** $[\text{K}^+] = 0.2\text{ M}, [\text{Ca}^{2+}] = 0.1\text{ M}, [\text{NO}_3^-] = 0.4\text{ M}$

**10.5d:**
*   **Calculation:** $V_{\text{tot}} = 300\text{ mL}$.
    $[\text{NaCl}] = 0.1/3\text{ M}$. $[\text{MgCl}_2] = 0.2/3\text{ M}$. $[\text{AlCl}_3] = 0.3/3\text{ M}$.
    $[\text{Cl}^-] = 1 \times (0.1/3) + 2 \times (0.2/3) + 3 \times (0.3/3) = \frac{0.1 + 0.4 + 0.9}{3} = \frac{1.4}{3} = 0.467\text{ M}$.
*   **Answer:** $[\text{Cl}^-] = 0.467\text{ M}$

</details>

### Type 6: Dilution of Ionic Solution

**The Pattern:** Diluting an ionic solution → all ion concentrations decrease proportionally.

#### Solved Example 10.6
**Q:** 100 mL of 0.5 M CaCl₂ is diluted to 500 mL. Find the new [Ca²⁺] and [Cl⁻]. 🟡

**Solution:**
```
Step 1: Find new [CaCl₂] after dilution
    M₁V₁ = M₂V₂
    0.5 × 100 = M₂ × 500
    M₂ = 0.1 M

Step 2: Ion concentrations
    CaCl₂ → Ca²⁺ + 2Cl⁻
    [Ca²⁺] = 0.1 M
    [Cl⁻] = 0.2 M

Answer: [Ca²⁺] = 0.1 M, [Cl⁻] = 0.2 M
```

> **Shortcut:** Each ion's concentration gets diluted by the same factor. Dilution factor = 500/100 = 5×. Original [Ca²⁺] = 0.5 M → new = 0.5/5 = 0.1 M. Original [Cl⁻] = 1.0 M → new = 1.0/5 = 0.2 M.

### Type 7: Ion Concentration After Mixing Two Electrolyte Solutions ⭐⭐

**The Pattern:** Combine mixing (Ch 9) with dissociation. Two different salts, possibly reacting.

#### Solved Example 10.7
**Q:** 100 mL of 0.2 M NaCl is mixed with 200 mL of 0.1 M Na₂SO₄. Find the concentration of Na⁺, Cl⁻, and SO₄²⁻. 🟡 ⭐

**Solution:**
```
Step 1: Find effective concentrations after mixing
    V_total = 100 + 200 = 300 mL

    [NaCl] = 0.2 × 100/300 = 0.0667 M
    [Na₂SO₄] = 0.1 × 200/300 = 0.0667 M

Step 2: Dissociation
    NaCl → Na⁺ + Cl⁻
        [Na⁺] from NaCl = 0.0667 M
        [Cl⁻] = 0.0667 M

    Na₂SO₄ → 2Na⁺ + SO₄²⁻
        [Na⁺] from Na₂SO₄ = 2 × 0.0667 = 0.1334 M
        [SO₄²⁻] = 0.0667 M

Step 3: Total [Na⁺] (common ion!)
    [Na⁺] = 0.0667 + 0.1334 = 0.2 M

Charge check:
    Positive: 0.2 × 1 = 0.2
    Negative: 0.0667 × 1 + 0.0667 × 2 = 0.0667 + 0.1334 = 0.2  ✅

Answer: [Na⁺] = 0.2 M, [Cl⁻] = 0.067 M, [SO₄²⁻] = 0.067 M
```

### Type 8: Precipitation — Will a Precipitate Form? (Ksp Preview) ⭐

**The Pattern:** When two electrolytes are mixed, an insoluble salt may form. Compare ionic product (Q) with Ksp.

#### Key Concept
```
Ionic Product (Q) = [cation]^a × [anion]^b  (after mixing, before precipitation)

If Q > Ksp → Precipitate forms ✅
If Q < Ksp → No precipitate ❌
If Q = Ksp → Saturated, at equilibrium
```

#### Solved Example 10.8
**Q:** 100 mL of 0.01 M BaCl₂ is mixed with 100 mL of 0.01 M Na₂SO₄. Will BaSO₄ precipitate? (Ksp of BaSO₄ = 1.1 × 10⁻¹⁰) 🟡 ⭐

**Solution:**
```
Step 1: Find ion concentrations after mixing
    V_total = 200 mL (each concentration halves)
    
    [Ba²⁺] = 0.01/2 = 0.005 M
    [SO₄²⁻] = 0.01/2 = 0.005 M

Step 2: Calculate ionic product (Q)
    Q = [Ba²⁺][SO₄²⁻] = 0.005 × 0.005 = 2.5 × 10⁻⁵

Step 3: Compare with Ksp
    Q = 2.5 × 10⁻⁵ >> Ksp = 1.1 × 10⁻¹⁰

    Q >> Ksp → BaSO₄ WILL precipitate! ✅

Answer: Yes, BaSO₄ precipitates (Q is about 10⁵ times larger than Ksp).
```

### Type 9: Weak Electrolytes — Partial Dissociation (α)

**The Pattern:** Weak electrolytes don't fully dissociate. Use degree of dissociation (α).

#### Key Formula
```
For HA ⇌ H⁺ + A⁻  (degree of dissociation = α)

    [H⁺] = C × α
    [A⁻] = C × α
    [HA] = C × (1 - α)

where C = initial concentration, α = fraction dissociated (0 < α < 1)
```

#### Solved Example 10.9
**Q:** 0.1 M acetic acid (CH₃COOH) has α = 0.013 (1.3% dissociation). Find [H⁺], [CH₃COO⁻], and [CH₃COOH]. 🟡

**Solution:**
```
    CH₃COOH ⇌ H⁺ + CH₃COO⁻

    [H⁺] = C × α = 0.1 × 0.013 = 1.3 × 10⁻³ M
    [CH₃COO⁻] = C × α = 1.3 × 10⁻³ M
    [CH₃COOH] = C × (1 - α) = 0.1 × 0.987 = 0.0987 M ≈ 0.1 M

    (Only 1.3% of the acid has dissociated — most remains intact.)

Answer: [H⁺] = [CH₃COO⁻] = 1.3 × 10⁻³ M, [CH₃COOH] ≈ 0.099 M
```

> **Key difference from Types 1-8:** Strong electrolytes: α = 1 (100% dissociation, use stoichiometry directly). Weak electrolytes: α < 1 (need to know α to find ion concentrations).

#### Practice Questions — Type 9

| # | Question | Difficulty |
|---|----------|------------|
| 10.9c | Why is the total ion concentration of 0.1 M acetic acid much less than that of 0.1 M HCl? | 🟡 |

<details>
<summary>💡 Detailed Solutions for Type 9</summary>

**10.9a:**
*   **Calculation:** $[\text{NH}_4^+] = [\text{OH}^-] = C\alpha = 0.01 \times 0.04 = 4 \times 10^{-4}\text{ M}$.
    $[\text{NH}_3] = C(1-\alpha) = 0.01 \times 0.96 = 0.0096\text{ M}$.
*   **Answer:** $[\text{NH}_4^+] = 4 \times 10^{-4}\text{ M}$, $[\text{OH}^-] = 4 \times 10^{-4}\text{ M}$, $[\text{NH}_3] = 0.0096\text{ M}$

**10.9b:**
*   **Calculation:** $K_a = C\alpha^2 \Rightarrow \alpha^2 = \frac{3.5 \times 10^{-4}}{0.05} = 70 \times 10^{-4}$.
    $\alpha = \sqrt{70} \times 10^{-2} \approx 8.37 \times 10^{-2} = 0.0837$.
    $[\text{H}^+] = C\alpha = 0.05 \times 0.0837 = 4.18 \times 10^{-3}\text{ M}$. (Or use $[\text{H}^+] = \sqrt{K_a C}$).
*   **Answer:** $\alpha = 0.0837$, $[\text{H}^+] = 4.18 \times 10^{-3}\text{ M}$

**10.9c:**
*   **Calculation:** HCl is a strong electrolyte ($\alpha = 1$) so it dissociates completely into $0.1\text{ M H}^+$ and $0.1\text{ M Cl}^-$. Acetic acid is a weak electrolyte ($\alpha \ll 1$) and barely dissociates, meaning there are far fewer ions in solution.
*   **Answer:** Because acetic acid is a weak electrolyte and only partially dissociates, while HCl dissociates completely.

</details>

### Type 10: Charge Balance — Electroneutrality

**The Pattern:** The total positive charge in solution must always equal the total negative charge.

#### Solved Example 10.10
**Q:** A solution contains 0.1 M NaCl and 0.2 M CaCl₂. Verify the charge balance. 🟡

**Solution:**
```
Dissociation:
    NaCl → Na⁺ + Cl⁻          [Na⁺] = 0.1, [Cl⁻] = 0.1
    CaCl₂ → Ca²⁺ + 2Cl⁻      [Ca²⁺] = 0.2, [Cl⁻] = 0.4

Total ions:
    [Na⁺] = 0.1 M
    [Ca²⁺] = 0.2 M
    [Cl⁻] = 0.1 + 0.4 = 0.5 M

Charge balance:
    Total positive charge = [Na⁺](+1) + [Ca²⁺](+2)
                         = 0.1 × 1 + 0.2 × 2 = 0.1 + 0.4 = 0.5

    Total negative charge = [Cl⁻](−1)
                         = 0.5 × 1 = 0.5

    Positive = Negative ✅ (electroneutrality maintained)
```

> **Why charge balance matters:** It's a powerful error-checking tool. If your calculated ion concentrations don't satisfy charge balance, you've made a mistake somewhere.

#### Practice Questions — Type 10

| # | Question | Difficulty |
|---|----------|------------|
| 10.10c | A solution has [Na⁺] = 0.2 M, [Ca²⁺] = 0.1 M, [Cl⁻] = 0.3 M, [SO₄²⁻] = y M. Find y. | 🟡 |

<details>
<summary>💡 Detailed Solutions for Type 10</summary>

**10.10a:**
*   **Calculation:** Positive charge $= [\text{K}^+] \times 1 + [\text{Mg}^{2+}] \times 2 = 0.3(1) + 0.1(2) = 0.5\text{ M}$.
    Negative charge $= [\text{Cl}^-] \times 1 = x(1) = x$.
    Charge balance: $x = 0.5$.
*   **Answer:** $x = 0.5$

**10.10b:**
*   **Calculation:** $[\text{Al}^{3+}] = 0.2\text{ M}$, $[\text{SO}_4^{2-}] = 0.3\text{ M}$.
    Positive charge $= 0.2 \times 3 = 0.6$. Negative charge $= 0.3 \times 2 = 0.6$. $0.6 = 0.6$ ✅.
*   **Answer:** Positive charge ($0.6$) = Negative charge ($0.6$).

**10.10c:**
*   **Calculation:** Positive charge $= [\text{Na}^+](1) + [\text{Ca}^{2+}](2) = 0.2 + 0.2 = 0.4$.
    Negative charge $= [\text{Cl}^-](1) + [\text{SO}_4^{2-}](2) = 0.3 + 2y$.
    $0.4 = 0.3 + 2y \Rightarrow 0.1 = 2y \Rightarrow y = 0.05$.
*   **Answer:** $y = 0.05$

    (d) Positive charge $= 3(0.06) = 0.18$; Negative charge $= 1(0.18) = 0.18$. They match!

| 10.J5 | 0.2 M CH₃COOH (Ka = 1.8 × 10⁻⁵) is mixed with 0.1 M HCl in equal volumes. Find [H⁺] in the mixture. (Consider common ion effect on acetic acid's dissociation.) ⭐⭐ | 🔴 |

<details>
<summary>💡 Detailed Solutions for JEE Mains Arena</summary>

**10.J1:**
*   **Calculation:**
    (a) Ions present initially: Ag⁺, NO₃⁻, Ba²⁺, Cl⁻.

    (c) Ag⁺ is limiting. $10\text{ mmol AgCl}$ forms.
    (d) $V_{\text{tot}} = 200\text{ mL}$.
        Spectators: $[\text{Ba}^{2+}] = 10/200 = 0.05\text{ M}$. $[\text{NO}_3^-] = 10/200 = 0.05\text{ M}$.
        Leftover Cl⁻ = $20 - 10 = 10\text{ mmol}$. $[\text{Cl}^-] = 10/200 = 0.05\text{ M}$.
*   **Answer:** (a) Ag⁺, NO₃⁻, Ba²⁺, Cl⁻, (b) Yes, (c) $10\text{ mmol}$, (d) $[\text{Ba}^{2+}] = 0.05\text{ M}, [\text{NO}_3^-] = 0.05\text{ M}, [\text{Cl}^-] = 0.05\text{ M}$

**10.J2:**
*   **Calculation:** Equal volumes $\Rightarrow$ concs halve.
    $[\text{Al}_2\text{(SO}_4\text{)}_3] = 0.05\text{ M}$, $[\text{Na}_3\text{PO}_4] = 0.05\text{ M}$.
    (a) $[\text{Al}^{3+}] = 2(0.05) = 0.1\text{ M}$. $[\text{SO}_4^{2-}] = 3(0.05) = 0.15\text{ M}$.
        $[\text{Na}^+] = 3(0.05) = 0.15\text{ M}$. $[\text{PO}_4^{3-}] = 0.05\text{ M}$.
    (b) $Q = [\text{Al}^{3+}][\text{PO}_4^{3-}] = (0.1)(0.05) = 0.005$. $Q (5 \times 10^{-3}) > K_{sp} (9.84 \times 10^{-21})$. It will precipitate.

**10.J3:**
*   **Calculation:** Osmotic pressure $\propto$ total particle concentration ($M \times i$).
    Glucose ($i=1$) $\rightarrow 0.1\text{ M}$. NaCl ($i=2$) $\rightarrow 0.2\text{ M}$. CaCl₂ ($i=3$) $\rightarrow 0.3\text{ M}$. AlCl₃ ($i=4$) $\rightarrow 0.4\text{ M}$.
*   **Answer:** Glucose $<$ NaCl $<$ CaCl₂ $<$ AlCl₃

**10.J4:**
*   **Calculation:** Charge balance: $+_e = -_e \Rightarrow 0.5(1) + 0.2(1) = 0.4(1) + x(2) \Rightarrow 0.7 = 0.4 + 2x \Rightarrow x = 0.15\text{ M}$.
    To verify synthesis from those salts:
    All K⁺ comes from KCl $\Rightarrow$ need $0.2\text{ M KCl}$. This provides $0.2\text{ M Cl}^-$.
    We need $0.4\text{ M Cl}^-$ total, so remaining $0.2\text{ M Cl}^-$ must come from NaCl $\Rightarrow$ need $0.2\text{ M NaCl}$.
    This provides $0.2\text{ M Na}^+$. We need $0.5\text{ M Na}^+$ total, so remaining $0.3\text{ M Na}^+$ must come from Na₂SO₄ $\Rightarrow$ need $0.15\text{ M Na}_2\text{SO}_4$.
    This provides $0.15\text{ M SO}_4^{2-}$. This exactly matches $x = 0.15\text{ M}$.
*   **Answer:** $x = 0.15$. Yes, it can be made using $0.2\text{ M NaCl}$, $0.2\text{ M KCl}$, and $0.15\text{ M Na}_2\text{SO}_4$.

**10.J5:**
*   **Calculation:** Equal volumes $\Rightarrow$ concentrations halve. $[\text{CH}_3\text{COOH}] = 0.1\text{ M}$, $[\text{HCl}] = 0.05\text{ M}$.

    For CH₃COOH $\rightleftharpoons$ H⁺ + CH₃COO⁻:
    $K_a = \frac{[\text{H}^+][\text{CH}_3\text{COO}^-]}{[\text{CH}_3\text{COOH}]} \approx \frac{(0.05 + y)(y)}{0.1 - y} \approx \frac{0.05 y}{0.1} = \frac{y}{2}$ (assuming $y$ is very small).
    $y/2 = 1.8 \times 10^{-5} \Rightarrow y = 3.6 \times 10^{-5}\text{ M}$.
    Total $[\text{H}^+] = 0.05 + 3.6 \times 10^{-5} \approx 0.05\text{ M}$. The dissociation of the weak acid is dramatically suppressed (Common Ion Effect).
*   **Answer:** $[\text{H}^+] \approx 0.05\text{ M}$

</details>

## Key Takeaways from Chapter 10

```
┌──────────────────────────────────────────────────────────────┐
│              IONIC CONCENTRATION SUMMARY                      │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  Rule: A_xB_y → x A^n+ + y B^m-                             │
│        [A] = x × C,  [B] = y × C                            │
│                                                               │
│  Strong electrolytes: 100% dissociation (α = 1)             │
│  Weak electrolytes: partial (use α)                          │
│                                                               │
│  Common ions: ADD contributions from all sources             │
│  e.g., NaCl + KCl → [Cl⁻] = [NaCl] + [KCl]                │
│                                                               │
│  Charge balance (ALWAYS holds):                               │
│    Σ(cation charge × conc) = Σ(anion charge × conc)         │
│                                                               │
│  Precipitation: Q vs Ksp                                      │
│    Q > Ksp → precipitate forms                               │
│    Q < Ksp → no precipitate                                  │
│                                                               │
│  Total ions per formula unit:                                 │
│    NaCl: 2 | CaCl₂: 3 | Al₂(SO₄)₃: 5                      │
│    → affects colligative properties (van't Hoff factor)      │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```
