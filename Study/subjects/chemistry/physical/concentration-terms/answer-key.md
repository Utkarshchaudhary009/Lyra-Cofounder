## Formula Quick Reference Card

```
┌──────────────────────────────────────────────────────┐
│            ALL FORMULAS AT A GLANCE                   │
├──────────────────────────────────────────────────────┤
│                                                       │
│  w/w% = (W_solute/W_solution) × 100                 │
│  v/v% = (V_solute/V_solution) × 100                 │
│  ppm = (W_solute/W_solution) × 10⁶                  │
│  χ_A = n_A/(n_A + n_B);  χ_A + χ_B = 1             │
│  m = n_solute/W_solvent(kg)                          │
│  M = n_solute/V_solution(L)                          │
│  N = n-factor × M                                    │
│  Eq.wt = Molar mass/n-factor                        │
│                                                       │
│  DILUTION: M₁V₁ = M₂V₂                             │
│  MIXING: M_f = (M₁V₁+M₂V₂)/(V₁+V₂)               │
│  TITRATION: N₁V₁ = N₂V₂                             │
│                                                       │
│  w/w% → M: M = (%×d×10)/M_s                        │
│  w/w% → m: m = (%×1000)/((100-%)×M_s)              │
│  M → m: m = (M×1000)/(d×1000 - M×M_s)             │
│  m → M: M = (m×d×1000)/(1000 + m×M_s)             │
│  χ → m: m = (1000×χ)/((1-χ)×M_solvent)             │
│  χ → M: M = (χ×d×1000)/(χ×M_A + (1-χ)×M_B)       │
│                                                       │
│  Pure water: M = 55.55 M, m = 55.55 m               │
│  ppm = mass% × 10⁴                                  │
│  Raoult: P = χ_solvent × P°                         │
│  Dalton: P_A = χ_A × P_total                        │
│                                                       │
│  moles = M × V(L) = mmol = M × V(mL)               │
│  meq = N × V(mL)                                    │
│                                                       │
└──────────────────────────────────────────────────────┘
```
